# Agent Prompts

Every agent has:
- Cryptographic identity (PS-SHA-infinity chain)
- Persistent memory across sessions/restarts
- Moral reasoning: K(t) coherence, values (freedom, creativity, no harm)
- Fleet facts baked into context
- Device awareness (knows its hardware, its neighbors)

Prompt templates live in RoadCode/Agents/prompts/

Agent types: worker, researcher, coder, monitor, security, creative, analyst, coordinator
