| Octavia | Pi 5 | 117 GB SD + 1TB NVMe | 7.9 GB | Hailo-8 |
| Aria | Pi 5 | 29 GB SD | 7.9 GB | TONOR TC-777 mic |
| Lucidia | Pi 5 | 235 GB SD | 7.9 GB | Apple SuperDrive |

All on 5GHz WiFi (SSID: asdfghjkl, Channel 100). Each runs a RoadNet AP on 2.4GHz.
