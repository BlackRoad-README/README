# Strategic Plans

## 5-Year Plan (2026-2031)

*"What if the AI you talked to yesterday was still working on your project while you slept?"*

### Year 1 (2026): Prove the Thesis — First revenue on $40/month infrastructure

| Quarter | Target | Key Milestones |
|---------|--------|----------------|
| Q2 (Apr-Jun) | $1,400/mo, 170 users | Wire Chat + Auth + RoadPay, launch AI API, ship Chat Pro |
| Q3 (Jul-Sep) | $5,000/mo, 500 users | Memory-powered chat, AI Identity Twin dashboard, mesh.js SDK |
| Q4 (Oct-Dec) | $15,000/mo, 1,000 users | Multi-tenant memory, Prism Console v1, first enterprise pilot |

### Year 2 (2027): Scale the Routing — $200K MRR

- H1: API Economy — OpenAI-compatible API, 50+ models, SOC 2 Type I
- H2: Memory Platform — 2048 compression, memory portability, enterprise memory
- 15,000 users. Seed round optional (profitable without it)

### Year 3 (2028): Session Boundary Collapses — $1M MRR

- BlackRoad Workspace replaces Notion + ChatGPT + Copilot
- Memory becomes the moat — users can't leave because agents know them

### Year 4 (2029): Creator Economy — $5M MRR

- RoadTube, Canvas Studio, Writing Portal
- 80-90% creator revenue share

### Year 5 (2030-31): Ambient AI Layer — $20M+ MRR, 1M users

- The OS becomes invisible
- IPO path or strategic acquisition

**Invariants:** cheap infra ($40/mo), commodity models, sovereign memory, costs don't scale with users.
