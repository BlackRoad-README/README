## Rules
- Colors are for shapes only (circles, squares, bars). NEVER for text.
- Text is always white (#f5f5f5) or gray with opacity hierarchy.
- No gradient text. No app logo. Use solid circles and squares in brand colors.
- All websites follow the blackroad-security.html design system.
